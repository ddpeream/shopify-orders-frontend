# Shopify Orders Frontend

Panel operativo del reto técnico de órdenes Shopify e inventario de materiales.

## Requisitos

- Node.js `^22.18.0` o `>=24.12.0`.
- Backend disponible en `http://localhost:3000/api`.

## Configuración inicial

```bash
npm install
copy .env.example .env
npm run dev
```

## Scripts

```bash
npm run dev
npm run format:check
npm run lint
npm run type-check
npm run test:unit
npm run test:coverage
npm run test:e2e
npm run build
npm run verify
```

La documentación completa se incorporará cuando finalicen las pantallas y la integración.
